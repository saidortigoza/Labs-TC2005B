¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación para el paradigma de orientado a objetos, mientras que JavaScript es un lenguaje cuyo propósito es la interacción web. Java es un lenguaje compilado mientras que JS es interpretado, sin embargo ambos utilizan casi la misma sintaxis para la mayoría de las situaciones.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getDate: devuelve los días del mes.

getDay: devuelve los días de la semana.

getFullYear: devuelve el año.

getHours: devuelve la hora.

getMinutes: devuelve los minutos.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

pop: elimina el último elemento agregado.

push: agrega un elemento al final del arreglo.

toString: convierte un arreglo en un string.

splice: inserta y elimina elementos en una posición específica del arreglo.

concat: concatena arreglos.

¿Cómo se declara una variable con alcance local dentro de una función?

Se utiliza la palabra reservada let.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Se corre el riesgo de que esa variable genere cambios dentro de otras funciones del código.
